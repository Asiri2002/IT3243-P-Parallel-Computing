
# IT3243 (P) Parallel Computing

## Overview

This repository contains practical programs, assignments, and examples developed for the **IT3243 (P) Parallel Computing** module.

The purpose of this module is to understand the concepts of parallel computing and implement parallel algorithms using different programming techniques and tools.

## Topics Covered

* Introduction to Parallel Computing
* Concurrency and Parallelism
* Performance Metrics and Speedup
* Amdahl's Law
* Parallel Programming Models
* MPI (Message Passing Interface)
* Process Communication
* Data Distribution
* Parallel Algorithms
* Load Balancing
* Synchronization Techniques
* Performance Analysis

## Technologies Used

* C Programming Language
* MPI Library
* GCC Compiler
* OpenMPI

## Practical Exercises

* MPI Initialization and Finalization
* Point-to-Point Communication
* Collective Communication
* Broadcast Operations
* Scatter and Gather Operations
* Reduction Operations
* Parallel Array Processing
* Parallel Matrix Operations
* Performance Measurement


## Execution

```bash
mpiexec -n number of process project name.exe
mpiexec -n 4 project1.exe
```

## Learning Outcomes

After completing this module, students will be able to:

* Understand parallel computing concepts.
* Develop parallel applications using MPI.
* Analyze performance improvements using multiple processors.
* Implement efficient parallel algorithms.
